# 🏥 Health Prediction Application

## 📌 Overview
This project is a Health Prediction System that predicts potential health conditions based on patient blood test results and integrates an external AI/ML-based Health API. The application demonstrates a full-stack implementation using Python for backend logic and a web-based frontend interface.

---

## 🚀 Features
- Patient health prediction using machine learning model
- Input form for blood test parameters
- Integration with external AI/ML health API
- Real-time prediction results
- Simple and user-friendly web interface
- Modular and scalable backend design
